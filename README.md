# PyPDFForm

## Introduction
**PyPDFForm** is a Python library for PDF form processing. It provides essential functionalities to interact with PDF forms, including:

- Inspecting what data a PDF form needs to be filled with  
- Filling a PDF form using a simple Python dictionary  
- Creating form fields on a PDF  
- Extracting pages from PDFs  
- Merging multiple PDF files
